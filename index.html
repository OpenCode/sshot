SSHot

A simple software to manage multiple SSH connections. Save connections, call connection.

Create a list of parameters for you SSH connection and use a double click on a table row to open directly a new connection on remote server.

To use the software please install

  • python-sqlite3
  • python-qt4
  • python-qt4-dev
  • openssh
  • xterm
  • sshpass
